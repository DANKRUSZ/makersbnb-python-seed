Makers Bnb

Makersbnb is the first group project on Software Engineering Fundamentals at Makers Academy.
This project aims to demonstrate the ability to create a full-stack web application with a relational
database, working asynchronously in teams.

Project by:
Claire Peng, David O'Donnell, Daniel Kruszewski, Onoura Wright
